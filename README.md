# 🦊 Planova

**Planova** is an iOS application designed to simplify and enrich travel organization. 

This project was conceived, designed, and developed by the **Foxy Team** during the **Apple Foundation Program (S4) 2025-2026** at the University of Naples Federico II.

## 🚀 Key Features

* 🗺️ **Destination Search:** Interactive exploration using MapKit.
* 📚 **Cultural Integration:** Detailed information about points of interest powered by the Wikipedia API.
* 💰 **Budget Management:** Built-in tools to keep track of travel expenses.
* 🤖 **AI Assistant:** Smart suggestions powered by foundational models (Gemini API).
* 🗂️ **Trip History:** Save and manage your past travel itineraries.

## 🛠️ Technologies Used

* **Language:** Swift
* **UI Framework:** SwiftUI
* **System Frameworks:** MapKit, Foundation
* **External APIs:** Wikipedia API, Google Gemini API
* **Design:** Sketch / Keynote for pitching and prototyping.

## 🎓 Project Context
This repository contains the final code delivered at the end of the **Apple Foundation Program (Cohort S4, 2025-2026)** training weeks. The program is based on the CBL (Challenge Based Learning) framework, which allowed us to identify a real-world problem and develop Planova as a technological solution, managing the entire app lifecycle: from ideation (sketching and UI design) to full code implementation.
